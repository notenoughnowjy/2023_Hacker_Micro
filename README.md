협업이 정말 중요하다고 느낌이 들었습니다. 처음 해커톤에서 팀이 배정되었을 때 저 말고 다른 두분은 개발을 아예 못한다고 하시고, 사전교육을 듣지도 않고
오셨기 때문에 할 수 있는게 없다고 하셨었습니다. 처음의 제가 '조금이라도 절망하지 않았다' 라고 한다면 거짓말입니다. 앞이 막막했고, 이거를 혼자 어떻게
다 해야할까? 라는 생각이 들었지만 그렇다고 '아예 못하겠다. 포기하자'도 아니었습니다. 이때까지의 제가 공부해왔던 것들이 있고 어떠한 것을 하든 이뤄낼
자신이 있었기 떄문입니다. 하지만 주제를 정하고 실제로 수행하면서 생각이 바뀌게 되었습니다. 처음부터 바뀌었다는 것이 아니라, 수행을 하면서 힘들었던 부분이
많았지만, 팀원들이 함께 해주면서 힘을 불어넣어 주셨고, 개인의 할일을 찾아서 해주셨습니다. 개발을 하면서 벽에 아주 많이 부딪혔지만 포기하지 않고 나아갔습니다.
